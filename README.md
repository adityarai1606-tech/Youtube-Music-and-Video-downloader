# YouTube Downloader (GUI)

A clean, modern Windows desktop application to download **YouTube videos** and
**YouTube Music playlists** in selectable quality using **yt-dlp**.

## Features
- Graphical interface (no command line needed)
- Download YouTube videos (480p / 720p / 1080p / 4K*)
- Download YouTube Music playlists (high-quality audio)
- Audio & Video mode toggle
- Automatic playlist folder creation
- Embedded thumbnails & metadata
- Portable single EXE (no Python required)

\* Resolution depends on source availability.

## Download
Go to the **Releases** section to download the latest Windows EXE.

- Windows 10 / 11
- Internet connection
